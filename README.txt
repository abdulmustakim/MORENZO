MARENZO Website Package
=======================
Brand: MARENZO
Tagline: Make Today Legendary
Type: Premium fashion brand showcase website with marketplace buying links.

Pages included:
1. index.html - Home page
2. shop.html - Shop / collection page
3. product.html - Product details page using product query id
4. about.html - About MARENZO page
5. lookbook.html - Style gallery page
6. contact.html - Contact page

Folders:
assets/css/style.css      Main design and responsive styling
assets/js/products.js     Product names, prices, images, sizes, colors, and marketplace links
assets/js/main.js         Animations, product rendering, filters, menu, product details
assets/images/            Placeholder product and lookbook SVG images
assets/videos/            Add your real MP4 product videos here

How to edit products:
Open assets/js/products.js and change:
- name
- price
- image
- fabric
- fit
- colors
- sizes
- description
- amazon link
- meesho link
- flipkart link
- shopsy link
- store link

How marketplace buttons work:
Each product card and product detail page has direct buttons for:
- Amazon
- Meesho
- Flipkart
- Shopsy
- Store
These links open in a new tab. Replace the current search links with your exact product links after listing your clothes.

How to add videos:
Put your MP4 files in assets/videos/ with these suggested names:
- brand-film.mp4
- shirt-fit.mp4
- tshirt-styling.mp4
- oversized-streetwear.mp4
- fabric-closeup.mp4
For product-specific videos, use product id names like:
- midnight-oversized.mp4
- ivory-premium-shirt.mp4

How to run:
Double-click index.html in your browser.
For best result, upload the full folder to GitHub Pages, Netlify, Vercel, or any hosting service.

Important:
The current photos are custom placeholder SVGs. Replace them with your real MARENZO shirt/T-shirt photos and videos for a true professional brand look.
